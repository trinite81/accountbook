# 가계부 — 복식부기 웹 앱

개인 재무를 복식부기(Double-Entry Bookkeeping) 방식으로 관리하는 웹 가계부입니다.

## 실행 방법

```bash
npm install
npm run dev        # 개발 서버 (http://localhost:5173)
npm run build      # 프로덕션 빌드
npm run preview    # 빌드 결과 미리보기
```

## 기술 스택

| 역할 | 라이브러리 |
|------|-----------|
| UI 프레임워크 | React 18 + TypeScript |

## 복식부기 사용법

### 핵심 개념

모든 거래는 반드시 **차변(Debit)** 과 **대변(Credit)** 이 동일한 금액으로 기록됩니다.

| 계정 유형 | 차변(Debit) | 대변(Credit) |
|---------|------------|-------------|
| 자산 | 증가 | 감소 |
| 부채 | 감소 | 증가 |
| 수익 | 감소 | 증가 |
| 비용 | 증가 | 감소 |

### 거래 예시

**급여 수령** (300만원 은행 입금)
- 차변: 은행계좌 +3,000,000 (자산 증가)
- 대변: 급여 +3,000,000 (수익 증가)

**식비 지출** (카드 사용 30,000원)
- 차변: 식비 +30,000 (비용 증가)
- 대변: 신용카드 +30,000 (자산 감소)

**카드값 납부** (은행에서 카드 결제 200,000원)
- 차변: 신용카드미결제 +200,000 (부채 감소)
- 대변: 은행계좌 +200,000 (자산 감소)

## 화면 구성

### 거래 입력 (`/`)
- 오늘 수입/지출/잔액 요약 카드
- 날짜 필터 (오늘/이번주/이번달/전체)
- 거래 추가/수정/삭제
- 수입(파란색), 지출(빨간색)으로 구분

### 분석 리포트 (`/reports`)
- 기간 탭: 일별/주별/월별/연별
- 전기 대비 증감률
- 수입 vs 지출 막대 그래프
- 지출 항목별 파이 차트
- 잔액 추이 라인 차트
- 계정별 상세 내역 테이블

### 계정 설정 (`/settings`)
- 계정 추가/수정/삭제
- 아이콘·색상 커스터마이징
- 계정 활성/비활성 토글
- 기본값 초기화
- 샘플 데이터 생성
- JSON/CSV 내보내기
- JSON 가져오기

## 데이터 관리

- 데이터는 브라우저 **localStorage**에 저장됩니다.
- **JSON 내보내기/가져오기**로 백업 및 복원이 가능합니다.
- **CSV 내보내기**로 Excel 등에서 열람할 수 있습니다.
- 처음 시작할 때 **샘플 데이터 생성** (설정 화면)으로 3개월치 예시 데이터를 넣어볼 수 있습니다.
